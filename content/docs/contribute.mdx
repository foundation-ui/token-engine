Please take a moment to review this document before submitting your first pull request. It is also strongly recommend that you check for open issues and pull requests to see if someone else is working on something similar.

<DocsSection>

#### Fork this repo

<DocsTextBlock>
  You can fork this repo by clicking the fork button in the top right corner of
  [this page](https://github.com/foundation-ui/token-engine).
</DocsTextBlock>

<br />

#### Clone on your local machine

<DocsCodePreview
  language="shell"
  code={`git clone https://github.com/foundation-ui/token-engine.git`}
/>

#### Navigate to project directory

<DocsCodePreview language="shell" code={`cd token-engine`} />

#### Create a new Branch

<DocsCodePreview language="shell" code={`git checkout -b new-branch`} />

#### Install dependencies

<DocsCodePreview language="shell" code={`pnpm install`} />

#### Run the project

<DocsCodePreview language="shell" code={`pnpm dev`} />

</DocsSection>
